Conversio
=========

Conversio is a PHP library that provides a simple infrastructure to create your own converters and to perform any conversion.

**WIP**

Installation
------------

Add `leodido/conversio` to your `composer.json`.

```json
{
   "require": {
       "leodido/conversio": "v0.1.0"
   }
}
```
